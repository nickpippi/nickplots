Diversity (Shannon index)

Two views. Groups per region: inside each gate, the count of each group and the Shannon of that mixture (H≈0 = pure region). Regions per group (relative): for each treatment, the diversity of the regions it occupies — spread over many regions → high H, concentrated in few → low H. H in nats, J = Pielou (0–1). Result in the plot table.

Best pairs (group separation)

Tests all numeric columns, pairwise, and ranks which two best split the groups of the chosen column. Sorts by Silhouette (visual separation in the scatter); the LDA acc. column is the classification reading. The result appears in the plot table and exports together with the "Descriptive table".

Threshold lines & regions

Lines to split the plot. Choose the type: Horizontal (give Y), Vertical (give X) or Slanted (X, Y and angle in degrees on the data scale). In "Edit on the plot" you can also drag the lines. Uses the X/Y/hue from the Plot tab.

Named regions (gating)

Draw closed regions (straight-edged, tiltable polygons), name them and drag them on the plot to adjust. Double-click an edge to insert a vertex; double-click or right-click a vertex to remove it (min. 3). Drag the name to reposition it. The live per-group count appears below. "Create column" generates a categorical column with each observation's region name. Uses the X/Y/hue from the Plot tab.

In edit mode: drag the vertices or the interior; double-click the body adds a vertex, on a vertex removes it.

To merge regions, give them the same name (e.g. two cells both become "low") — they are counted together and get the same label in the created column.

Save regions + lines to a file to reuse on another sheet (with the same axes). It warns if you load them on different axes.

Export gating — each cell is assigned to a single region (the one it fits best: deeper "inside" wins; on the line goes to the nearest). The Region column shows which. Keeps all columns. Check which ones to include.
